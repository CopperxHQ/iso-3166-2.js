## Negative Keywords (Avoid)

These keywords attract wrong audience:
- `country code phone` (telephone country codes)
- `dialing code` (phone)
- `calling code` (phone)
- `airport code` (IATA codes)
- `currency code` (ISO 4217, different standard)
